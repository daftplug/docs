### Add Your API Key

<Tabs>
  <Tab title="OpenAI (ChatGPT)">
    1. Navigate to **Generatify â†’ Settings â†’ Integrations**
    2. Find the **OpenAI** section
    3. Toggle **Enable OpenAI** to ON
    4. Enter your OpenAI API key
    5. Click **Save Settings**

    <Accordion title="Where to get an OpenAI API key?">
      1. Visit [platform.openai.com](https://platform.openai.com)
      2. Sign up or log in to your account
      3. Go to **API Keys** section
      4. Click **Create new secret key**
      5. Copy the key (you won't see it again!)
      6. Paste it into Generatify settings

      **Pricing:** Pay-as-you-go based on usage. Most features cost fractions of a cent per request.
    </Accordion>
  </Tab>

  <Tab title="Anthropic (Claude)">
    1. Navigate to **Generatify â†’ Settings â†’ Integrations**
    2. Find the **Anthropic** section
    3. Toggle **Enable Anthropic** to ON
    4. Enter your Anthropic API key
    5. Click **Save Settings**

    <Accordion title="Where to get an Anthropic API key?">
      1. Visit [console.anthropic.com](https://console.anthropic.com)
      2. Sign up or log in to your account
      3. Go to **API Keys** section
      4. Click **Create Key**
      5. Copy the key
      6. Paste it into Generatify settings

      **Pricing:** Pay-as-you-go. Claude models are competitive with OpenAI pricing.
    </Accordion>
  </Tab>

  <Tab title="Google (Gemini)">
    1. Navigate to **Generatify â†’ Settings â†’ Integrations**
    2. Find the **Google Gemini** section
    3. Toggle **Enable Google Gemini** to ON
    4. Enter your Google AI API key
    5. Click **Save Settings**

    <Accordion title="Where to get a Google AI API key?">
      1. Visit [makersuite.google.com](https://makersuite.google.com/app/apikey)
      2. Sign in with your Google account
      3. Click **Get API Key**
      4. Create a new API key
      5. Copy the key
      6. Paste it into Generatify settings

      **Pricing:** Generous free tier available. Pay-as-you-go for higher volumes.
    </Accordion>
  </Tab>
</Tabs>

## Common Questions

<AccordionGroup>
  <Accordion title="Do I need to choose just one AI provider?">
    No! You can connect multiple AI providers and assign different models to different features. For example:
    - Use GPT-4 for your AI Assistant (most powerful)
    - Use GPT-3.5 Turbo for auto-tagging (faster, cheaper)
    - Use Claude for helpdesk (great at conversation)
  </Accordion>

  <Accordion title="How much will AI API calls cost?">
    It depends on your usage, but most features cost just fractions of a cent per action:
    - **Auto tags:** ~$0.001 per post
    - **Chatbot message:** ~$0.01 - $0.05 per conversation
    - **Helpdesk response:** ~$0.01 - $0.03 per question

    Most sites spend $5-20/month total. Monitor your provider's dashboard to track exact costs.
  </Accordion>

  <Accordion title="Can I use Generatify without API keys?">
    Some features like **Toxicity Checker** and **Alt Image Detection** work entirely client-side using TensorFlow.js and don't require API keys. However, most features require an AI provider connection.
  </Accordion>

  <Accordion title="Is my data sent to AI providers?">
    Yes, when you use AI features, relevant content (post text, user questions, etc.) is sent to your chosen AI provider for processing. Data is sent securely via HTTPS. No data is stored by Generatify itselfâ€”everything goes directly to your AI provider.
  </Accordion>

  <Accordion title="Can I enable all features at once?">
    Yes! You can enable as many features as you want. Each module operates independently. Just keep in mind that more features = more API calls = higher costs.
  </Accordion>
</AccordionGroup>
