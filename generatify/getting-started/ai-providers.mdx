## OpenAI (ChatGPT)

The most widely-used AI provider, powering ChatGPT. Known for high-quality outputs and strong reasoning capabilities.

### Available Models

<Tabs>
  <Tab title="GPT-4 Turbo">
    **Best for:** Complex tasks, high-quality content, troubleshooting

    - Most capable model
    - Excellent reasoning and comprehension
    - Handles complex instructions well
    - Best for AI Assistant and Helpdesk Agent

    **Cost:** ~$0.01 per 1,000 input tokens, ~$0.03 per 1,000 output tokens
  </Tab>

  <Tab title="GPT-3.5 Turbo">
    **Best for:** Automations, high-volume tasks, cost-effective use

    - Fast and affordable
    - Great for simpler tasks
    - Ideal for auto-tagging and categorization
    - Good balance of quality and cost

    **Cost:** ~$0.0005 per 1,000 input tokens, ~$0.0015 per 1,000 output tokens
  </Tab>

  <Tab title="GPT-4o Mini">
    **Best for:** Ultra-fast responses, budget-friendly tasks

    - Fastest OpenAI model
    - Very affordable
    - Good for quick suggestions and simple queries

    **Cost:** ~$0.00015 per 1,000 input tokens, ~$0.0006 per 1,000 output tokens
  </Tab>
</Tabs>

### Setup Instructions

<Steps>
  <Step title="Get an API Key">
    1. Visit [platform.openai.com](https://platform.openai.com)
    2. Sign up or log in
    3. Navigate to **API Keys**
    4. Click **Create new secret key**
    5. Copy the key (you won't see it again!)
  </Step>

  <Step title="Add to Generatify">
    1. Go to **Generatify â†’ Settings â†’ Integrations**
    2. Find the **OpenAI** section
    3. Toggle **Enable OpenAI** to ON
    4. Paste your API key
    5. Click **Save Settings**
  </Step>

  <Step title="Verify Connection">
    Look for a green checkmark next to OpenAI indicating successful connection.
  </Step>
</Steps>

### Pricing & Billing

- **Pay-as-you-go** based on token usage
- Tokens â‰ˆ words (1,000 tokens â‰ˆ 750 words)
- Add billing details at [platform.openai.com/account/billing](https://platform.openai.com/account/billing)
- Set usage limits to control costs
- Monitor usage at [platform.openai.com/usage](https://platform.openai.com/usage)

<Tip>
  **Cost Optimization:** Use GPT-4 for quality-critical tasks and GPT-3.5 Turbo for high-volume automations. You can assign different models to different features in Generatify.
</Tip>

## Anthropic (Claude)

Created by former OpenAI researchers, Claude excels at natural conversation and follows instructions carefully with built-in safety features.

<Tabs>

    - Natural, human-like responses

    - Great for customer-facing features

## Google (Gemini)

Google's latest AI models with a generous free tier, multimodal capabilities, and excellent speed.

### Available Models

<Tabs>
  <Tab title="Gemini 1.5 Pro">
    **Best for:** Complex analysis, long content, document understanding

    - Massive context window (1M tokens)
    - Strong reasoning capabilities
    - Great for analyzing long documents
    - Multimodal (text and images)

    **Cost:** Free tier available, then ~$0.00125 per 1,000 input tokens
  </Tab>

  <Tab title="Gemini 1.5 Flash">

## Security Best Practices

<Warning>
  **API keys are sensitive!** Treat them like passwords. Never share them publicly or commit them to version control.
</Warning>

### Protecting Your API Keys

<AccordionGroup>
  <Accordion title="Use Environment Variables" icon="lock">
    For developers, store API keys in environment variables instead of hardcoding:

    ```php
    // wp-config.php
    define('GENERATIFY_OPENAI_KEY', getenv('OPENAI_API_KEY'));
    define('GENERATIFY_ANTHROPIC_KEY', getenv('ANTHROPIC_API_KEY'));
    define('GENERATIFY_GOOGLE_KEY', getenv('GOOGLE_API_KEY'));
    ```
  </Accordion>

  <Accordion title="Set Usage Limits" icon="gauge">
    Prevent unexpected charges by setting limits in your AI provider dashboard:

    - **OpenAI:** Set monthly budget limits
    - **Anthropic:** Configure spending alerts
    - **Google:** Enable quota alerts
  </Accordion>

  <Accordion title="Monitor Usage Regularly" icon="chart-line">
    Check your usage dashboards weekly:
    - Look for unexpected spikes
    - Identify which features use the most tokens
    - Adjust model selection if costs are too high
  </Accordion>

  <Accordion title="Rotate Keys Periodically" icon="arrows-rotate">
    For security, rotate your API keys every few months:
    1. Generate a new key in your provider dashboard
    2. Update Generatify with the new key
    3. Delete the old key from your provider
  </Accordion>
</AccordionGroup>

## Troubleshooting Connection Issues

<AccordionGroup>
  <Accordion title="Red X / Connection Failed">
    **Possible causes:**
    - Invalid or expired API key
    - API key doesn't have required permissions
    - Network/firewall blocking requests
    - HTTPS not enabled on your site

    **Solutions:**
    1. Double-check you copied the entire API key
    2. Verify the key is active in your provider dashboard
    3. Ensure your WordPress site uses HTTPS
    4. Check if server firewall blocks external API calls
  </Accordion>

  <Accordion title="API Key Works Then Stops">
    **Possible causes:**
    - Billing issue (credit card expired, payment failed)
    - Usage limits reached
    - API key was revoked or regenerated

    **Solutions:**
    1. Check billing status in provider dashboard
    2. Verify payment method is valid
    3. Check usage limits haven't been exceeded
    4. Generate a new key if needed
  </Accordion>

  <Accordion title="Slow Response Times">
    **Possible causes:**
    - Using a slower model
    - Server latency
    - High provider traffic

    **Solutions:**
    1. Switch to faster models (Flash, Haiku, 3.5 Turbo)
    2. Check your server's internet connection
    3. Try a different AI provider temporarily
  </Accordion>

  <Accordion title="'Quota Exceeded' Error">
    **Possible causes:**
    - Free tier limits reached (Google)
    - Rate limits exceeded
    - Monthly quota consumed

    **Solutions:**
    1. Wait for quota to reset (usually daily/monthly)
    2. Upgrade to paid tier
    3. Switch to a different provider temporarily
    4. Reduce feature usage or adjust settings
  </Accordion>
</AccordionGroup>
