## Why Meta Descriptions Matter

<AccordionGroup>
  <Accordion title="They Appear in Search Results" icon="google">
    Meta descriptions are the snippet text below your page title in Google search results. They're often the deciding factor in whether someone clicks your link.
    
    **Impact:**
    - 5-10% CTR improvement with good descriptions
    - Direct influence on traffic
    - First impression of your content
  </Accordion>
  
  <Accordion title="Google Uses Them for Ranking" icon="chart-line">
    While not a direct ranking factor, meta descriptions affect:
    - Click-through rate (which IS a ranking factor)
    - User engagement signals
    - Page relevance perception
    
    **Result:** Better descriptions → More clicks → Higher rankings
  </Accordion>
  
  <Accordion title="Social Sharing" icon="share-nodes">
    Meta descriptions appear when your content is shared:
    - Facebook posts
    - Twitter cards
    - LinkedIn shares
    - Messaging apps
    
    **Benefit:** Professional appearance on social media
  </Accordion>
  
  <Accordion title="They're Time-Consuming to Write" icon="clock">
    Manual meta description writing:
    - 3-5 minutes per post
    - Requires SEO knowledge
    - Easy to forget
    - Boring to write
    
    **Solution:** Let AI handle it automatically!
  </Accordion>
</AccordionGroup>

## How It Works

<Steps>
  <Step title="Content Analysis">
    AI reads post title, content, excerpt, and focus keywords
  </Step>
  
  <Step title="Key Point Extraction">
    Identifies main topics, value propositions, and unique angles
  </Step>
  
  <Step title="Description Generation">
    Creates 150-160 character compelling description
  </Step>
  
  <Step title="SEO Optimization">
    Ensures keywords are included naturally
  </Step>
  
  <Step title="Length Validation">
    Confirms it fits Google's display limits
  </Step>
  
  <Step title="Automatic Insertion">
    Adds to post meta (Yoast, Rank Math, or WordPress SEO field)
  </Step>
</Steps>

<Info>
**Optimal Length:** AI generates 150-160 characters—the sweet spot that displays fully in search results without truncation.
</Info>

## Setup Guide

### Enable Meta Descriptions

<Steps>
  <Step title="Navigate to Settings">
    Go to **WordPress Admin → Generatify → Settings → Automations**
  </Step>
  
  <Step title="Find Meta Descriptions Section">
    Scroll to **Auto-Generate Meta Descriptions**
  </Step>
  
  <Step title="Enable Feature">
    Toggle **Enable Meta Descriptions** to ON
  </Step>
  
  <Step title="Configure Settings">
    Set your preferred options (see Configuration below)
  </Step>
  
  <Step title="Save Settings">
    Click **Save Settings**
  </Step>
</Steps>

## Advanced Customization

### Custom Description Prompts

Tailor AI descriptions to your brand voice:

```
Navigate to: Generatify → Settings → Automations → Meta Descriptions → Advanced

Default Prompt:
"Write a compelling SEO meta description (155 characters) for this post. 
Include the focus keyword naturally and a call to action."

Custom Example (E-commerce):
"Write a 155-character meta description emphasizing benefits, urgency, 
and value. Include the product name and a strong call to action like 
'Shop now' or 'Order today'. Use power words: exclusive, limited, premium."

Custom Example (News Site):
"Write a 155-character meta description in AP style. Lead with the news hook, 
include key facts, and maintain objectivity. No call to action."
```

### Developer Hook

Modify descriptions programmatically:

```php
add_filter('generatify_meta_description', function($description, $post_id) {
    $post = get_post($post_id);
    
    // Add brand name suffix for all descriptions
    $brand = 'TechBlog';
    if (strlen($description) + strlen($brand) + 3 <= 160) {
        $description .= " | $brand";
    }
    
    // Remove certain phrases
    $description = str_replace(['Click here', 'Read more'], 'Learn', $description);
    
    // Ensure proper capitalization
    $description = ucfirst($description);
    
    return $description;
}, 10, 2);
```

## Troubleshooting

<AccordionGroup>
  <Accordion title="Description Too Long/Short">
    **Symptoms:** Descriptions exceed 160 chars or are under 140
    
    **Solutions:**
    1. Adjust "Target Length" setting (try 152-158)
    2. AI will get closer to target over time
    3. Regenerate if far off
    4. Switch to GPT-4o for better precision
    5. Check custom prompt isn't forcing length issues
  </Accordion>
  
  <Accordion title="Description Not Appearing in Search">
    **Symptoms:** Google shows different text
    
    **Reasons:**
    1. Google sometimes chooses different text
    2. This is normal—Google rewriting descriptions is common
    3. Happens when Google thinks other text is more relevant
    
    **Solutions:**
    1. Make description more relevant to search query
    2. Ensure it matches page content accurately
    3. Wait—Google may start using it over time
    4. Check robots.txt isn't blocking crawling
  </Accordion>
  
  <Accordion title="Not Saving to SEO Plugin">
    **Symptoms:** Description generates but doesn't save
    
    **Solutions:**
    1. Verify SEO plugin is installed and active
    2. Check Generatify has detected plugin (see Settings)
    3. Update SEO plugin to latest version
    4. Check for plugin conflicts (disable other SEO tools)
    5. Try manual generation to see error messages
  </Accordion>
  
  <Accordion title="Generic/Boring Descriptions">
    **Symptoms:** AI writes bland, template-like descriptions
    
    **Solutions:**
    1. Switch to Claude 3.5 Sonnet (more creative)
    2. Change tone to "Engaging"
    3. Customize AI prompt with specific guidelines
    4. Ensure post content has clear value proposition
    5. Add more context in post excerpt
  </Accordion>
  
  <Accordion title="Wrong Focus Keyword Used">
    **Symptoms:** AI uses incorrect keyword in description
    
    **Solutions:**
    1. Set focus keyword in your SEO plugin first
    2. Make sure keyword appears in post title/content
    3. Disable "Include Keywords" if forcing wrong terms
    4. Customize prompt to emphasize specific keywords
  </Accordion>
</AccordionGroup>
