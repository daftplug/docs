## Filters

Filters allow you to modify data before it's processed or displayed.

### AI Prompts & Responses

<AccordionGroup>
  <Accordion title="generatify_ai_prompt" icon="message">
    **Modify AI prompts before sending to API**
    
    ```php
    apply_filters( 'generatify_ai_prompt', string $prompt, string $feature, array $context )
    ```
    
    **Parameters:**
    - `$prompt` (string): The prompt text
    - `$feature` (string): Feature name (chat, tags, categories, meta_description, etc.)
    - `$context` (array): Additional context data
    
    **Returns:** Modified prompt string
    
    **Example:**
    ```php
    add_filter( 'generatify_ai_prompt', function( $prompt, $feature, $context ) {
        // Add custom instructions for chat
        if ( $feature === 'chat' ) {
            $prompt .= "\n\nAlways end responses with a friendly emoji.";
        }
        
        // Add brand voice for all features
        $prompt .= "\n\nUse professional, friendly tone.";
        
        return $prompt;
    }, 10, 3 );
    ```
    
    **Use Cases:**
    - Add brand voice guidelines
    - Include custom instructions per feature
    - Add context-specific requirements
  </Accordion>
  
  <Accordion title="generatify_ai_response" icon="reply">
    **Filter AI responses after receiving from API**
    
    ```php
    apply_filters( 'generatify_ai_response', string $response, string $feature, array $context )
    ```
    
    **Example:**
    ```php
    add_filter( 'generatify_ai_response', function( $response, $feature ) {
        // Remove unwanted phrases
        $remove = [
            'As an AI',
            'As a language model',
            'I don\'t have personal opinions',
        ];
        $response = str_replace( $remove, '', $response );
        
        // Add custom suffix for chat
        if ( $feature === 'chat' ) {
            $response .= "\n\n_Need more help? [Contact us](/contact)_";
        }
        
        return $response;
    }, 10, 2 );
    ```
  </Accordion>
  
  <Accordion title="generatify_ai_model" icon="brain">
    **Change AI model dynamically**
    
    ```php
    apply_filters( 'generatify_ai_model', string $model, string $feature, array $context )
    ```
    
    **Example:**
    ```php
    add_filter( 'generatify_ai_model', function( $model, $feature, $context ) {
        // Use GPT-4 for important pages
        if ( $feature === 'meta_description' && $context['post_id'] === 123 ) {
            return 'gpt-4';
        }
        
        // Use Claude for chat on product pages
        if ( $feature === 'chat' && is_singular( 'product' ) ) {
            return 'claude-3-5-sonnet-20241022';
        }
        
        return $model;
    }, 10, 3 );
    ```
  </Accordion>
  
  <Accordion title="generatify_ai_parameters" icon="sliders">
    **Modify API request parameters**
    
    ```php
    apply_filters( 'generatify_ai_parameters', array $params, string $feature )
    ```
    
    **Example:**
    ```php
    add_filter( 'generatify_ai_parameters', function( $params, $feature ) {
        // Increase creativity for marketing content
        if ( $feature === 'meta_description' ) {
            $params['temperature'] = 0.9;
        }
        
        // More deterministic for categorization
        if ( $feature === 'categories' ) {
            $params['temperature'] = 0.3;
        }
        
        // Limit response length
        $params['max_tokens'] = 500;
        
        return $params;
    }, 10, 2 );
    ```
  </Accordion>
</AccordionGroup>

### Automations

<AccordionGroup>
  <Accordion title="generatify_generated_tags" icon="tags">
    **Modify auto-generated tags**
    
    ```php
    apply_filters( 'generatify_generated_tags', array $tags, int $post_id, array $context )
    ```
    
    **Example:**
    ```php
    add_filter( 'generatify_generated_tags', function( $tags, $post_id ) {
        // Always add specific tag
        if ( ! in_array( 'Featured', $tags ) ) {
            $tags[] = 'Featured';
        }
        
        // Remove unwanted tags
        $blacklist = [ 'spam', 'test', 'draft' ];
        $tags = array_diff( $tags, $blacklist );
        
        // Limit to 5 tags
        $tags = array_slice( $tags, 0, 5 );
        
        // Convert to lowercase
        $tags = array_map( 'strtolower', $tags );
        
        return $tags;
    }, 10, 2 );
    ```
  </Accordion>
  
  <Accordion title="generatify_generated_categories" icon="folder">
    **Modify auto-generated categories**
    
    ```php
    apply_filters( 'generatify_generated_categories', array $categories, int $post_id )
    ```
    
    **Example:**
    ```php
    add_filter( 'generatify_generated_categories', function( $categories, $post_id ) {
        // Map AI categories to your taxonomy
        $mapping = [
            'Tech' => 'Technology',
            'Business' => 'Business & Finance',
            'Marketing' => 'Digital Marketing',
        ];
        
        $mapped_categories = [];
        foreach ( $categories as $category ) {
            $mapped_categories[] = $mapping[ $category ] ?? $category;
        }
        
        // Always assign "Uncategorized" if no categories
        if ( empty( $mapped_categories ) ) {
            $mapped_categories[] = 'Uncategorized';
        }
        
        return $mapped_categories;
    }, 10, 2 );
    ```
  </Accordion>
  
  <Accordion title="generatify_meta_description" icon="file-lines">
    **Customize generated meta descriptions**
    
    ```php
    apply_filters( 'generatify_meta_description', string $description, int $post_id )
    ```
    
    **Example:**
    ```php
    add_filter( 'generatify_meta_description', function( $description, $post_id ) {
        // Add brand name
        $description .= ' | Acme Corp';
        
        // Ensure length limit (160 chars)
        if ( strlen( $description ) > 160 ) {
            $description = substr( $description, 0, 157 ) . '...';
        }
        
        // Add emoji for specific categories
        $categories = get_the_category( $post_id );
        foreach ( $categories as $category ) {
            if ( $category->slug === 'tutorials' ) {
                $description = 'ðŸ“š ' . $description;
                break;
            }
        }
        
        return $description;
    }, 10, 2 );
    ```
  </Accordion>
  
  <Accordion title="generatify_automation_content" icon="file">
    **Modify content before AI processing**
    
    ```php
    apply_filters( 'generatify_automation_content', string $content, int $post_id, string $feature )
    ```
    
    **Example:**
    ```php
    add_filter( 'generatify_automation_content', function( $content, $post_id, $feature ) {
        // Strip shortcodes
        $content = strip_shortcodes( $content );
        
        // Remove images
        $content = preg_replace( '/<img[^>]+>/i', '', $content );
        
        // For tagging, only send first 500 words
        if ( $feature === 'tags' ) {
            $content = wp_trim_words( $content, 500, '' );
        }
        
        return $content;
    }, 10, 3 );
    ```
  </Accordion>
</AccordionGroup>

### Chat & Helpdesk

<AccordionGroup>
  <Accordion title="generatify_chat_context" icon="message-lines">
    **Add context to chat conversations**
    
    ```php
    apply_filters( 'generatify_chat_context', array $context, int $user_id, string $session_id )
    ```
    
    **Example:**
    ```php
    add_filter( 'generatify_chat_context', function( $context, $user_id ) {
        // Add user information
        $user = get_userdata( $user_id );
        $context['user_info'] = [
            'name' => $user->display_name,
            'email' => $user->user_email,
            'registration_date' => $user->user_registered,
        ];
        
        // Add recent orders (WooCommerce)
        if ( function_exists( 'wc_get_orders' ) ) {
            $orders = wc_get_orders( [
                'customer_id' => $user_id,
                'limit' => 5,
            ] );
            $context['recent_orders'] = array_map( function( $order ) {
                return [
                    'id' => $order->get_id(),
                    'total' => $order->get_total(),
                    'status' => $order->get_status(),
                ];
            }, $orders );
        }
        
        return $context;
    }, 10, 2 );
    ```
  </Accordion>
  
  <Accordion title="generatify_knowledge_sources" icon="book">
    **Add custom knowledge sources**
    
    ```php
    apply_filters( 'generatify_knowledge_sources', array $sources, array $context )
    ```
    
    **Example:**
    ```php
    add_filter( 'generatify_knowledge_sources', function( $sources, $context ) {
        // Add external API data
        if ( $context['page_type'] === 'product' ) {
            $product_id = $context['product_id'];
            $api_data = fetch_product_specs_from_api( $product_id );
            
            $sources[] = [
                'type' => 'external_api',
                'content' => $api_data,
                'weight' => 1.5, // Prioritize this source
            ];
        }
        
        // Add custom post type
        $faqs = get_posts( [
            'post_type' => 'faq',
            'posts_per_page' => 20,
        ] );
        
        foreach ( $faqs as $faq ) {
            $sources[] = [
                'type' => 'faq',
                'title' => $faq->post_title,
                'content' => $faq->post_content,
                'url' => get_permalink( $faq->ID ),
            ];
        }
        
        return $sources;
    }, 10, 2 );
    ```
  </Accordion>
  
  <Accordion title="generatify_chat_display_rules" icon="eye">
    **Control when chat widget appears**
    
    ```php
    apply_filters( 'generatify_chat_display_rules', bool $should_display, array $context )
    ```
    
    **Example:**
    ```php
    add_filter( 'generatify_chat_display_rules', function( $should_display, $context ) {
        // Hide on checkout page
        if ( is_checkout() ) {
            return false;
        }
        
        // Only show to logged-in users
        if ( ! is_user_logged_in() ) {
            return false;
        }
        
        // Show only during business hours (9 AM - 5 PM EST)
        $hour = (int) current_time( 'H' );
        if ( $hour < 9 || $hour > 17 ) {
            return false;
        }
        
        return $should_display;
    }, 10, 2 );
    ```
  </Accordion>
</AccordionGroup>

        // Exclude specific categories
        $args['category__not_in'] = [ 5, 12, 18 ];
        
        // Require featured image
        $args['meta_query'] = [

                'compare' => 'EXISTS',

### Content Moderation

<AccordionGroup>
  <Accordion title="generatify_toxicity_threshold" icon="shield-check">
    **Dynamically adjust toxicity thresholds**
    
    ```php
    apply_filters( 'generatify_toxicity_threshold', int $threshold, string $category, int $user_id )
    ```
    
    **Example:**
    ```php
    add_filter( 'generatify_toxicity_threshold', function( $threshold, $category, $user_id ) {
        // More strict for new users
        $comment_count = get_comments( [
            'user_id' => $user_id,
            'status' => 'approve',
            'count' => true,
        ] );
        
        if ( $comment_count < 3 ) {
            $threshold -= 20; // Lower threshold = more strict
        }
        
        // More lenient outside business hours
        $hour = (int) current_time( 'H' );
        if ( $hour < 8 || $hour > 18 ) {
            $threshold += 10;
        }
        
        // Zero tolerance for threats
        if ( $category === 'threat' ) {
            $threshold = 50; // Very strict
        }
        
        return $threshold;
    }, 10, 3 );
    ```
  </Accordion>
  
  <Accordion title="generatify_moderation_action" icon="gavel">
    **Change moderation action based on context**
    
    ```php
    apply_filters( 'generatify_moderation_action', string $action, array $toxicity_scores, array $comment_data )
    ```
    
    **Possible actions:** `hold`, `block`, `warn`, `approve`
    
    **Example:**
    ```php
    add_filter( 'generatify_moderation_action', function( $action, $toxicity_scores, $comment_data ) {
        // Block if multiple categories triggered
        $triggered_count = count( array_filter( $toxicity_scores, function( $score ) {
            return $score > 60;
        } ) );
        
        if ( $triggered_count >= 3 ) {
            return 'block';
        }
        
        // Always approve for admins
        $user = get_user_by( 'email', $comment_data['comment_author_email'] );
        if ( $user && user_can( $user, 'manage_options' ) ) {
            return 'approve';
        }
        
        return $action;
    }, 10, 3 );
    ```
  </Accordion>
</AccordionGroup>

## Actions

Actions allow you to execute code at specific points in the plugin's lifecycle.

### AI Requests

<AccordionGroup>
  <Accordion title="generatify_before_ai_request" icon="arrow-right">
    **Before sending request to AI API**
    
    ```php
    do_action( 'generatify_before_ai_request', string $feature, array $data )
    ```
    
    **Example:**
    ```php
    add_action( 'generatify_before_ai_request', function( $feature, $data ) {
        // Log API usage
        error_log( sprintf( 
            '[Generatify] AI request: %s for post %d',
            $feature,
            $data['post_id'] ?? 0
        ) );
        
        // Track analytics
        if ( function_exists( 'gtag' ) ) {
            ?>
            <script>
            gtag('event', 'ai_request', {
                'feature': '<?php echo esc_js( $feature ); ?>'
            });
            </script>
            <?php
        }
        
        // Update usage counter
        $usage = get_option( 'generatify_usage_count', [] );
        $usage[ $feature ] = ( $usage[ $feature ] ?? 0 ) + 1;
        update_option( 'generatify_usage_count', $usage );
    }, 10, 2 );
    ```
  </Accordion>
  
  <Accordion title="generatify_after_ai_request" icon="check">
    **After receiving response from AI API**
    
    ```php
    do_action( 'generatify_after_ai_request', string $feature, mixed $response, array $data )
    ```
    
    **Example:**
    ```php
    add_action( 'generatify_after_ai_request', function( $feature, $response, $data ) {
        // Log response
        error_log( sprintf(
            '[Generatify] AI response received for %s: %d tokens',
            $feature,
            $data['tokens_used'] ?? 0
        ) );
        
        // Update cost tracking
        $cost = calculate_ai_cost( $feature, $data['tokens_used'] );
        $total_cost = get_option( 'generatify_total_cost', 0 );
        update_option( 'generatify_total_cost', $total_cost + $cost );
        
        // Send to analytics
        wp_remote_post( 'https://analytics.example.com/track', [
            'body' => json_encode( [
                'event' => 'ai_response',
                'feature' => $feature,
                'tokens' => $data['tokens_used'],
            ] ),
        ] );
    }, 10, 3 );
    ```
  </Accordion>
  
  <Accordion title="generatify_ai_request_error" icon="triangle-exclamation">
    **When AI API request fails**
    
    ```php
    do_action( 'generatify_ai_request_error', string $feature, WP_Error $error, array $data )
    ```
    
    **Example:**
    ```php
    add_action( 'generatify_ai_request_error', function( $feature, $error, $data ) {
        // Send alert email
        wp_mail(
            'admin@example.com',
            'Generatify API Error',
            sprintf(
                "Feature: %s\nError: %s\nPost ID: %d",
                $feature,
                $error->get_error_message(),
                $data['post_id'] ?? 0
            )
        );
        
        // Log to custom error table
        global $wpdb;
        $wpdb->insert( 'generatify_errors', [
            'feature' => $feature,
            'error' => $error->get_error_message(),
            'data' => json_encode( $data ),
            'created_at' => current_time( 'mysql' ),
        ] );
    }, 10, 3 );
    ```
  </Accordion>
</AccordionGroup>
