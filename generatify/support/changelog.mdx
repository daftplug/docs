## Updating Generatify

### Automatic Updates (Recommended)

<Steps>
  <Step title="Enable Auto-Updates">
    Navigate to: `Plugins ‚Üí Installed Plugins`
    
    Find Generatify, click **Enable auto-updates**
  </Step>
  
  <Step title="WordPress Handles It">
    WordPress will automatically update Generatify in the background
    
    - Checks for updates daily
    - Installs updates during off-peak hours
    - Sends email notification after update
  </Step>
  
  <Step title="Review Changelog">
    After update, review changelog:
    
    - Admin notice with "What's New" button
    - Or visit: Settings ‚Üí About ‚Üí Changelog
  </Step>
</Steps>

### Manual Updates

<Steps>
  <Step title="Backup First">
    **Always backup before updating:**
    
    - Database backup (phpMyAdmin, plugin, hosting control panel)
    - Files backup (FTP, hosting control panel)
    - Or use backup plugin (UpdraftPlus, BackWPup)
  </Step>
  
  <Step title="Check for Updates">
    Navigate to: `Dashboard ‚Üí Updates`
    
    Look for Generatify in plugin updates list
  </Step>
  
  <Step title="Update Plugin">
    Click **Update Now** next to Generatify
    
    - WordPress downloads new version
    - Installs over old version
    - Activates automatically
  </Step>
  
  <Step title="Verify">
    Check that everything works:
    
    - Visit Settings ‚Üí About ‚Üí Version (should show new version)
    - Test key features (chat, automations)
    - Check for any error notices
  </Step>
</Steps>

<Warning>
**Important:** Always backup your site before updating plugins. While we test extensively, conflicts with your specific setup are possible.
</Warning>

## Breaking Changes & Migration

We strive to maintain backwards compatibility, but occasionally breaking changes are necessary.

### v1.0.0 ‚Üí v1.0.1
**No breaking changes.** Update safely.

### Future Breaking Changes

When breaking changes occur, we provide:
- **3 months advance notice** (in changelog and email)
- **Migration guide** with step-by-step instructions
- **Compatibility mode** to continue using old behavior temporarily
- **Support** to help with migration

<Info>
**Semantic Versioning:** Generatify follows [semantic versioning](https://semver.org/):
- **Major** (2.0.0): Breaking changes
- **Minor** (1.1.0): New features, backwards-compatible
- **Patch** (1.0.1): Bug fixes, backwards-compatible
</Info>
