## Overview

Anthropic's Claude models are known for natural conversation, strong safety features, and excellent instruction following. Created by former OpenAI researchers, Claude excels at helpdesk and customer-facing interactions.

## Setup Guide

### Step 1: Create Anthropic Account

<Steps>
  <Step title="Visit Anthropic Console">
    Go to [console.anthropic.com](https://console.anthropic.com)
  </Step>

  <Step title="Sign Up">
    Create an account with email
  </Step>

  <Step title="Verify Email">
    Check inbox and verify your email address
  </Step>

  <Step title="Add Payment Method">
    Navigate to **Settings → Billing** and add payment details
  </Step>
</Steps>

<Info>
  **New Account Credit:** Anthropic often provides $5 free credit for new accounts to get started.
</Info>

### Step 2: Generate API Key

<Steps>
  <Step title="Navigate to API Keys">
    In Anthropic Console, go to **Settings → API Keys**
  </Step>

  <Step title="Create New Key">
    Click **Create Key**
  </Step>

  <Step title="Name Your Key">
    Give it a descriptive name (e.g., "Generatify WordPress")
  </Step>

  <Step title="Copy Key">
    Copy the API key immediately
  </Step>
</Steps>

<img src="/images/anthropic-api-key.png" alt="Creating Anthropic API Key" />

### Step 3: Add to Generatify

<Steps>
  <Step title="Open WordPress Admin">
    Navigate to **Generatify → Settings → Integrations**
  </Step>

  <Step title="Find Anthropic Section">
    Scroll to the **Anthropic** integration block
  </Step>

  <Step title="Enable Anthropic">
    Toggle **Enable Anthropic** to ON
  </Step>

  <Step title="Paste API Key">
    Paste your Anthropic API key
  </Step>

  <Step title="Save Settings">
    Click **Save Settings**
  </Step>

  <Step title="Verify Connection">
    Look for green checkmark confirming connection
  </Step>
</Steps>

## Cost Management

### Understanding Pricing

**Token Basics:**
- Input tokens: What you send to Claude
- Output tokens: What Claude generates
- Both count toward costs
- ~750 words = 1,000 tokens

**Example Calculation:**
```
Task: Helpdesk response
Input tokens: ~500 (user question + knowledge base context)
Output tokens: ~150 (Claude's response)

Cost with Claude 3.5 Sonnet:
Input: 500 × $3.00 / 1M = $0.0015
Output: 150 × $15.00 / 1M = $0.00225
Total: ~$0.00375 per response
```

### Setting Budget Alerts

<Steps>
  <Step title="Go to Billing Settings">
    Navigate to **Settings → Billing** in Anthropic Console
  </Step>

  <Step title="Set Monthly Budget">
    Define your maximum monthly spend
  </Step>

  <Step title="Configure Alerts">
    Get emails at 50%, 75%, and 90% of budget
  </Step>

  <Step title="Enable Auto-cutoff">
    Optionally stop usage at budget limit
  </Step>
</Steps>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Authentication Error">
    **Error:** "Invalid API key"

    **Solutions:**
    1. Verify key copied correctly
    2. Check key is active in console
    3. Ensure billing is configured
    4. Generate new key if needed
  </Accordion>

  <Accordion title="Rate Limit Exceeded">
    **Error:** "Rate limit reached"

    **Solutions:**
    1. Wait 60 seconds and retry
    2. Check your tier limits
    3. Implement request queuing
    4. Upgrade tier with more spend
  </Accordion>

  <Accordion title="Overloaded Error">
    **Error:** "Overloaded"

    **Solutions:**
    1. This is temporary—retry in a few seconds
    2. Implement exponential backoff
    3. Try different model temporarily
    4. Check [status.anthropic.com](https://status.anthropic.com)
  </Accordion>

  <Accordion title="Response Refused">
    **Issue:** Claude refuses to answer

    **Reasons:**
    - Question involves harmful content
    - Request violates usage policies
    - Prompt is ambiguous or unclear

    **Solutions:**
    1. Rephrase question more clearly
    2. Remove potentially harmful elements
    3. Review Anthropic usage policies
  </Accordion>
</AccordionGroup>
