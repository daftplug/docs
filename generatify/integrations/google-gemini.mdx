## Overview

Google's Gemini models offer cutting-edge AI capabilities with a generous free tier, making them perfect for getting started or scaling affordably. Known for speed, multimodal capabilities, and massive context windows.

## Setup Guide

### Step 1: Get Google AI API Key

<Steps>
  <Step title="Visit Google AI Studio">
    Go to [aistudio.google.com](https://aistudio.google.com)
  </Step>

  <Step title="Sign In">
    Sign in with your Google account
  </Step>

  <Step title="Get API Key">
    Click **Get API Key** in the top navigation
  </Step>

  <Step title="Create New Key">
    Click **Create API Key** and choose a Google Cloud project (or create new)
  </Step>

  <Step title="Copy Key">
    Copy the generated API key
  </Step>
</Steps>

<img src="/images/google-api-key.png" alt="Creating Google AI API Key" />

<Info>
  **No Credit Card Required:** You can start using Gemini's free tier immediately without adding payment details!
</Info>

### Step 2: Add to Generatify

<Steps>
  <Step title="Open WordPress Admin">
    Navigate to **Generatify ‚Üí Settings ‚Üí Integrations**
  </Step>

  <Step title="Find Google Section">
    Scroll to the **Google Gemini** integration block
  </Step>

  <Step title="Enable Google Gemini">
    Toggle **Enable Google Gemini** to ON
  </Step>

  <Step title="Paste API Key">
    Paste your Google AI API key
  </Step>

  <Step title="Save Settings">
    Click **Save Settings**
  </Step>

  <Step title="Verify Connection">
    Look for green checkmark confirming connection
  </Step>
</Steps>

## Upgrading to Paid Tier

When you exceed free tier limits:

<Steps>
  <Step title="Set Up Billing">
    In Google AI Studio or Google Cloud Console, add billing account
  </Step>

  <Step title="Enable Paid API">
    Switch from free tier to pay-as-you-go pricing
  </Step>

  <Step title="Set Budget Alerts">
    Configure spending alerts in Google Cloud Console
  </Step>

  <Step title="Monitor Usage">
    Track usage in Google Cloud Console dashboards
  </Step>
</Steps>

<Info>
  **Gradual Transition:** You can continue using free tier alongside paid usage. Google charges only for requests that exceed free limits.
</Info>

**Example Calculation:**
```
Task: Auto-tag a blog post
Input tokens: ~300 (title + content excerpt)
Output tokens: ~20 (5 tags)

Cost with Gemini 1.5 Flash:
Input: 300 √ó $0.075 / 1M = $0.0000225
Output: 20 √ó $0.30 / 1M = $0.000006
Total: ~$0.00003 per post

1,000 posts = ~$0.03 (vs. $1-2 with other providers!)
```

## Troubleshooting

<AccordionGroup>
  <Accordion title="API Key Not Working">
    **Error:** "API key not valid"

    **Solutions:**
    1. Verify key copied correctly (no spaces)
    2. Check API is enabled in Google Cloud Console
    3. Ensure you're using AI Studio API key (not Cloud API key)
    4. Try generating a new key
  </Accordion>

  <Accordion title="Free Tier Limit Reached">
    **Error:** "Quota exceeded"

    **Solutions:**
    1. Wait for quota to reset (per minute/day limits)
    2. Upgrade to paid tier for higher limits
    3. Optimize feature usage
    4. Use Flash instead of Pro (higher free limits)
  </Accordion>

  <Accordion title="Billing Not Enabled">
    **Error:** "Billing must be enabled"

    **Solutions:**
    1. This means you hit free tier limits
    2. Add billing in Google Cloud Console
    3. Or wait for free tier reset
    4. Reduce usage to stay within free tier
  </Accordion>

  <Accordion title="Slow Responses">
    **Issue:** Gemini feels slow

    **Solutions:**
    1. Switch to Gemini 1.5 Flash (faster)
    2. Reduce context/prompt length
    3. Check your internet connection
    4. Try different time of day (off-peak)
  </Accordion>
</AccordionGroup>
