## Choosing the Right Provider

<AccordionGroup>
  <Accordion title="Choose OpenAI if..." icon="openai">
    - You need the most powerful models available
    - Content quality is your top priority
    - You're familiar with ChatGPT
    - Budget allows for premium pricing
    - You need GPT-4's advanced reasoning
    - Vision capabilities are important
  </Accordion>

  <Accordion title="Choose Anthropic if..." icon="a">
    - You're building a helpdesk or chatbot
    - Natural conversation is critical
    - Safety and content filtering matter
    - You want competitive pricing
    - Long context windows are needed
    - You value ethical AI development
  </Accordion>

  <Accordion title="Choose Google if..." icon="google">
    - You want to start with free tier
    - Budget is a primary concern
    - You need fast responses
    - High volume usage expected
    - Multimodal features interest you
    - You prefer Google ecosystem
  </Accordion>
</AccordionGroup>

## Security & Privacy

<Warning>
  API keys are sensitive credentials. Never share them publicly or commit them to version control.
</Warning>

### Best Practices

- Store API keys in WordPress database (encrypted)
- Use environment variables for additional security
- Rotate keys every 3-6 months
- Set spending limits in provider dashboards
- Monitor usage for unusual activity
- Revoke compromised keys immediately

### Data Privacy

When you use Generatify:
- Content is sent to your chosen AI provider for processing
- Data transmission uses HTTPS encryption
- Providers may temporarily cache requests for processing
- Review each provider's privacy policy for data retention

**Provider Privacy Policies:**
- [OpenAI Privacy Policy](https://openai.com/privacy)
- [Anthropic Privacy Policy](https://www.anthropic.com/privacy)
- [Google AI Privacy](https://support.google.com/gemini/answer/13594961)

## Troubleshooting

<AccordionGroup>
  <Accordion title="API key not working">
    1. Verify key is copied correctly (no extra spaces)
    2. Check key is active in provider dashboard
    3. Ensure billing is set up
    4. Verify API access is enabled
    5. Try generating a new key
  </Accordion>

  <Accordion title="Rate limit errors">
    1. Check your tier limits in provider dashboard
    2. Reduce feature usage temporarily
    3. Upgrade to higher tier if needed
    4. Spread requests over time
    5. Consider using multiple providers
  </Accordion>

  <Accordion title="High unexpected costs">
    1. Check usage dashboard for spike
    2. Review which features are active
    3. Look for loops or repeated calls
    4. Set budget alerts
    5. Switch to cheaper models
  </Accordion>
</AccordionGroup>
