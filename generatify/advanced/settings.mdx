## Import/Export Settings

### Backup Configuration

<Steps>
  <Step title="Go to Settings">
    Navigate to **Generatify → Settings → Advanced**
  </Step>
  
  <Step title="Export Settings">
    Click **Export Settings** button
  </Step>
  
  <Step title="Download JSON">
    Save `generatify-settings.json` file
  </Step>
</Steps>

---

### Restore Configuration

<Steps>
  <Step title="Go to Settings">
    Navigate to **Generatify → Settings → Advanced**
  </Step>
  
  <Step title="Import Settings">
    Click **Import Settings** button
  </Step>
  
  <Step title="Upload JSON">
    Select your `generatify-settings.json` file
  </Step>
  
  <Step title="Confirm">
    Review changes and click **Apply Settings**
  </Step>
</Steps>

<Warning>
**Note:** Importing settings will overwrite all current configuration. API keys are not included in exports for security.
</Warning>

## Default Reset

Reset all settings to defaults:

```
Settings → Advanced → Reset

⚠️ Reset Options:
- Reset all settings
- Reset specific feature (e.g., just AI Assistant)
- Keep API keys

Action: [Reset to Defaults]
```

<Warning>
**Irreversible:** Always export settings before resetting.
</Warning>
